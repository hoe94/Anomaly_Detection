Anomaly Detection for Maybank (1155) Bursa Stock price in Covid Period

Project Description

In this project, we built a ML model to perform the Anomaly Detection on Maybank stock price in Covid period (18th March 2020 - Now). The objective of this project is, to capture the anomaly/outlier by using Isolation Forest Algorithm. We tuned the model to assume 10% data are anomalies among the data. User can observed that the red markers represents the anomaly/outlier in the line graph. Besides, user can check the closing price on the selected date. And the Percentage_Change% column describes the difference of stock price within 2 days in percentage%. For example, Closing stock price for 28/4 is RM9.04, and the stock price is RM9.07. Which means that it increase 0.33% on the stock price.
